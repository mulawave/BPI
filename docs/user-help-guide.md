# BPI User Help & Support Guide

This guide explains the user experience in simple terms. Use it whenever you need help with any part of the app.

## 1) Getting started
- Create an account on the Register page. You will enter name, screen name, gender, email, password, confirm password, and the simple math captcha.
- Accept the terms and register.
- Log in on the Login page using your email and password.
- After login, you land on the dashboard.

## 2) Dashboard overview (your home screen)
The dashboard is the main hub for all user activities.

### 2.1 Portfolio overview
- Shows Total Portfolio Value.
- You can hide/show balances.
- You see a wallet breakdown for Main, Palliative, and BPT.
- A 24h change indicator shows gain or loss over time.

### 2.2 Quick actions
- Deposit: add funds to your wallet.
- Withdraw: move funds out.
- Transfer: send funds to others.

### 2.3 Wallets and balances
- Primary wallets include Main, Locked Capital, Rewards, and BPT.
- Community wallets include Palliative, Shelter, Community, Health, Education, etc.
- You can review balances and see detailed activity in wallet timelines.

### 2.4 Notifications and alerts
- Notifications appear in the Notifications center.
- You can open the Notifications modal to see recent alerts.

### 2.5 Profile completion
- You may be prompted to complete your profile.
- You can update details like name, phone, and address.
- Location and bank details can be entered using guided fields.

### 2.6 Taxes & referrals
- Taxes summary can be opened from the Taxes modal.
- Referrals can be reviewed from the Referral details modal.

### 2.7 Community and growth tools
The dashboard includes community features and growth tools such as:
- Community updates
- Calculator
- Deals
- Leadership pool
- EPC/EPP resources
- Solar assessment
- Training center
- Promotional materials
- Submit channel and browse channels
- Third‑party opportunities

### 2.8 Palliative journey
- You can view the Palliative journey and activation progress.
- A palliative activation modal guides steps if required.

### 2.9 Profile security
- Access Security settings to enable a 4‑digit PIN.
- Enable 2FA (Google Authenticator) if available.

## 3) Store
The Store is where you browse, purchase, and manage product pickups.

### 3.1 Browse and search
- Browse featured and all products.
- Filter by product type and token.
- Search by name or description.

### 3.2 Product details
- View product images, description, price, and rewards.
- See accepted tokens and payment caps.
- Start checkout or save for later.

### 3.3 Checkout
- Review order details.
- Choose payment mode:
  - Fiat only
  - Hybrid (token + fiat)
  - Token only (if allowed)
- Confirm checkout to create your order.

### 3.4 Orders & claim codes
- Open My Store Orders to track progress.
- When ready, your claim code appears.
- Copy the claim code and present it at the pickup center.

### 3.5 Pickup confirmation and rating
- After staff verifies your claim, confirm pickup in your orders screen.
- Rate your pickup experience and submit feedback.

### 3.6 Pickup centers
- Browse available pickup centers.
- View location, contact name, phone, and status.

## 4) Blog
The Blog helps you stay informed.

### 4.1 Blog home
- See featured posts in a hero carousel.
- Search posts by keyword.
- Filter by category.
- Read latest headlines.

### 4.2 Blog post page
- Read the full article with author and stats.
- Add a comment and join the discussion.
- View comments and pagination.

## 5) CSP (Community Support Program)
This is where you request and track support.

### 5.1 Eligibility
- You must meet membership, direct referrals, and contribution rules.
- The dashboard shows your eligibility status.

### 5.2 Submit a request
- Choose a category (National or Global).
- Enter amount, purpose, and notes.
- Submit for admin approval.

### 5.3 Live status
- Track broadcast time remaining.
- View raised amount and contributors.

### 5.4 Extensions and splits
- See time extension rules (paid or referral-based).
- See how contributions are split across wallets.

### 5.5 History
- View recent support requests and outcomes.

## 6) Account settings
Settings are accessible from the dashboard.

### 6.1 Security
- Enable or update your 4‑digit PIN.
- Enable 2FA if the system has it enabled.
- Disable 2FA with PIN + code if required.

### 6.2 Other tabs
Other tabs (Profile, Notifications, Preferences, Privacy, Billing) may show “Coming Soon”.

## 7) Troubleshooting
- Can’t log in: check your email, password, and internet connection.
- Forgot password: use the “Forgot Password” link on login.
- No claim code yet: your order may still be processing.
- Checkout error: open the store again and restart checkout.
- 2FA issues: verify your code and ensure time is synced in your authenticator app.

## 8) Getting help
- Check this guide first.
- Use the Help Center link inside Settings.
- Contact support with your email and a short description of the issue.
